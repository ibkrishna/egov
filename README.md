
# eGov Landing Page — Task 1 & Task 2

This project is a recreation of the **eGov Foundation landing page**, built as part of a technical assignment. It demonstrates both static frontend development and modern React-based UI implementation.

## Tech Stack

Task 1
Framework: HTML, CSS, Bootstrap 5
Font: DM Sans (Google Fonts)
Icons: Bootstrap Icons
Data: Static HTML

Task 2
Framework: React 18
Font: DM Sans (Google Fonts)
Icons: Unicode / Emoji
Data: Mock JSON (Simulated API)

## Task 1 — HTML + CSS + Bootstrap

A fully responsive static landing page built using Bootstrap 5.

### How to Run

No setup required — just open the file:

bash
cd task1

start index.html  # Windows


(For macOS: `open index.html` | Linux: `xdg-open index.html`)

### Features

* Responsive Navbar
* Hero Section
* Stats Banner
* Bold Approaches Section
* Sustaining Change Section
* DIGIT Call-To-Action
* Latest at eGov Section
* Supporters Section
* Footer

### Bonus

* Smooth hover animations
* CSS transitions
* Mobile-friendly layout

## Task 2 — React Application


A dynamic version of the landing page using React with modern best practices.

### How to Run

bash
cd task2
npm install
npm start


Open in browser:
[http://localhost:3000](http://localhost:3000)

## Features

* Functional Components with React Hooks

  * useState
  * useEffect
  * useMemo
* Simulated API call with loading state (1.2s delay)
* Search functionality (filter by title/description)
* Category-based filtering (tabs like What's New, Articles, Whitepapers, etc.)
* "No Results Found" state handling
* Fully responsive UI
* Interactive hover animations
